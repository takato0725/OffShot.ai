# OffShot.ai

**AI美女の日常を、洗練されたプロンプトに昇華する。**

Z-image / Wan 両対応の、アイドル日常スナップ風プロンプト自動生成ツールです。
ボタン1つで「雑に撮れたiPhone写真」っぽいプロンプトを生成します。

---

## ダウンロード・使い方

### 方法① GitHubからダウンロード（推奨）

1. このページ右上の緑の **「Code」** ボタンをクリック
2. **「Download ZIP」** を選択
3. ZIPを解凍する
4. フォルダ内の **`index.html`** をブラウザにドラッグ＆ドロップ

```
解凍したフォルダ/
└── index.html  ← これをブラウザで開くだけ
```

### 方法② ファイルを直接保存

1. [`index.html`](./index.html) を開く
2. ページ上で右クリック → 「名前を付けて保存」
3. 保存した `index.html` をブラウザで開く

> **インストール不要・サーバー不要・APIキー不要** です。
> ファイルをダブルクリックするだけで動きます。

---

## 使い方

1. **パラメータを設定** — 18種のカテゴリから各項目を選ぶ（ランダムのままでもOK）
2. **🔒 ロックで固定** — 気に入った項目は鍵アイコンで固定
3. **プロンプト生成** ボタンを押す
4. **Z-Image** / **Wan Video** / **和訳** タブを選んでプロンプトをコピー

---

## 機能

- **ワンクリック生成** — ボタンを押すだけでプロンプトを生成
- **Z-Image / Wan Video 切り替え** — 静止画・動画モデルに合わせたプロンプトを表示
- **和訳タブ** — 生成されたプロンプトの日本語訳を表示
- **18種の組み込みパラメータ** — 下表を参照
- **衣装サブオプション** — カラー（13色）と柄（10種）を個別に選択可能
- **ロック機能** — 気に入ったパラメータを固定してその他だけ変動
- **ランダム変動数スライダー** — 毎回変える項目数を1〜10で調整
- **カスタムカテゴリ** — 独自カテゴリを追加・編集・削除（localStorage保存）
- **カスタム直接入力** — 各パラメータに自分のテキストを直接入力
- **生成履歴** — 最大8件を保存、クリックで復元
- **自動コピー** — 2回目以降の生成で自動的にクリップボードへコピー
- **ダークモード** — ライト/ダーク切替対応

## プロンプト出力形式

生成されるプロンプトは **3セクション構成** です：

```
{シーン系パラメータ}  【hair: {髪型}. outfit: {服装}】  {テクニカル系パラメータ + 品質タグ}
```

| セクション | 含まれるパラメータ |
|---|---|
| **冒頭（シーン）** | body, expression, makeup, location, time_of_day, distance, angle, pose, held_item, emotion |
| **中央【】（外見）** | hair + accessory, outfit + color + pattern |
| **末尾（テクニカル）** | exposure, noise, motion, 品質・スタイルタグ |

セクション間はダブルスペースで区切られ、外見パラメータは `【】` 全角括弧でラベル付けされます。
