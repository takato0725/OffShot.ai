# OffShot.ai

**AI美女の日常を、プロンプトに。**

Z-image / Wan 両対応の、アイドル日常スナップ風プロンプト自動生成ツールです。
ボタン1つで「雑に撮れたiPhone写真」っぽいプロンプトを生成します。

---

## ✨ 機能

- **ワンクリック生成** — ボタンを押すだけでプロンプトを生成
- **Z-image / Wan 切り替え** — 同画面でモデルに合わせたプロンプトを表示
- **8種のパラメータ** — ロケーション・距離感・アングル・露出・髪型・服装・ノイズ・ブレ
- **ロック機能** — 気に入ったパラメータを固定してその他だけ変動させる
- **ランダム変動数スライダー** — 毎回変える項目数を2〜7で調整
- **カスタムカテゴリ** — 表情・小物など独自カテゴリを追加・編集・削除（localStorage保存）
- **カスタム直接入力** — 各パラメータに自分のテキストを直接入力してプロンプトに反映
- **生成履歴** — 最大8件を保存、クリックで復元

## 🛠 技術スタック

- 純粋な HTML / CSS / JavaScript（ビルドツール不要）
- 外部依存なし（Google Fonts のみ CDN 使用）
- APIキー不要・オフライン動作
- localStorage でカスタムカテゴリを永続化
