# OffShot.ai

**AI美女の日常を、プロンプトに。**

Z-image / Wan 両対応の、アイドル日常スナップ風プロンプト自動生成ツールです。
ボタン1つで「雑に撮れたiPhone写真」っぽいプロンプトを生成します。

---

## ダウンロード・使い方

### 方法① GitHubからダウンロード（推奨）

1. このページ右上の緑の **「Code」** ボタンをクリック
2. **「Download ZIP」** を選択
3. ZIPを解凍する
4. フォルダ内の **`index.html`** をブラウザにドラッグ＆ドロップ

```
解凍したフォルダ/
└── index.html  ← これをブラウザで開くだけ
```

### 方法② ファイルを直接保存

1. [`index.html`](./index.html) を開く
2. ページ上で右クリック → 「名前を付けて保存」
3. 保存した `index.html` をブラウザで開く

> **インストール不要・サーバー不要・APIキー不要** です。
> ファイルをダブルクリックするだけで動きます。

---

## 使い方

1. **パラメータを設定** — ロケーション・体型・ポーズなど各項目を選ぶ（ランダムのままでもOK）
2. **🔒 ロックで固定** — 気に入った項目は鍵アイコンで固定
3. **GENERATE PROMPT** ボタンを押す
4. **Z-image** または **Wan** タブを選んでプロンプトをコピー

---

## ✨ 機能

- **ワンクリック生成** — ボタンを押すだけでプロンプトを生成
- **Z-image / Wan 切り替え** — 同画面でモデルに合わせたプロンプトを表示
- **11種の組み込みパラメータ** — 下表を参照
- **ロック機能** — 気に入ったパラメータを固定してその他だけ変動
- **ランダム変動数スライダー** — 毎回変える項目数を2〜7で調整
- **カスタムカテゴリ** — 表情・小物など独自カテゴリを追加・編集・削除（localStorage保存）
- **カスタム直接入力** — 各パラメータに自分のテキストを直接入力
- **生成履歴** — 最大8件を保存、クリックで復元

| 距離感 | 6種 | 超クローズアップ〜全身・望遠圧縮 |
| アングル | 5種 | 目線・ローアングル・ハイアングル・後ろ姿 など |
| 露出・色味 | 5種 | ナチュラル・夕方オレンジ・蛍光灯・夜暗め など |
| 髪型 | 10種 | ポニーテール・ボブ・ツインテール・濡れ髪 など |
| 服装 | 13種 | カジュアル・ビキニ・浴衣・スポーツブラ など |
| ノイズ感 | 4種 | クリア〜フィルム粒子感 |
| ブレ・ピンボケ | 4種 | シャープ〜被写体ブレ・ピンボケ |
| 体型 | 5種 | スレンダー・グラマー・スポーティ・小柄 など |
| ポーズ | 7種 | 立ち姿・座り・歩き途中・壁もたれ・振り返り など |
| 手に持っているもの | 8種 | スマホ・ペットボトル・傘・タオル など |
| 時間帯 | 7種 | 早朝・朝・昼・夕方・夜・深夜 |
